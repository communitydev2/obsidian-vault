How to Write a SMART Goal:

1. Start with a vague goal: "I want to get fit". 
    

  

Make it Specific: "I want to run 3 miles without stopping". 

  

Make it Measurable: "I will run 3 miles without stopping, tracked by my fitness app". 

  

Make it Achievable: "I will build up to 3 miles by running 3 times a week". 

  

Make it Relevant: "This goal is important because it improves my cardiovascular health". 

  

Make it Time-bound: "I will achieve this within 3 months (by [Date])".
